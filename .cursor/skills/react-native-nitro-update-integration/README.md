# react-native-nitro-update integration skill

This Cursor skill guides integration of `react-native-nitro-update` into any React Native app.

**Use in this repo:** The agent can use it when you ask to add OTA or integrate the library.

**Use in another project:** The same skill is installed in your **personal** skills at `~/.cursor/skills/react-native-nitro-update-integration/`. When you open your other app in Cursor, say e.g. “Help me add react-native-nitro-update” or “Follow the react-native-nitro-update integration skill” and the agent will apply the steps.

**Full integration doc:** `packages/react-native-nitro-update/INTEGRATION.md`
